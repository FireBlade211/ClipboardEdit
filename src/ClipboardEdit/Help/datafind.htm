To find data in the current clipboard format

  1. Click the Find option on the Edit menu.
  2. In the Find dialog, select whether to find text or hex data using the tabs at the top.
  3. Type the data you want to find in the box shown.
  4. Click Find Next to find the next occurence of the data.
Notes
  • The search will start at your current selection and move forward only.
  • When searching text, you can check the Match Case check box to only find text that matches the exact casing you specify.
  • To search text, the text must be ANSI. ClipboardEdit cannot find Unicode text, unless you use the manual Hex data search option.
