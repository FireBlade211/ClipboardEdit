Documentation Index Generator

The documentation index generator is what generates the documentation index whenever ClipboardEdit is launched for the first time or the Update Documentation button is clicked on the menu bar.

When the documentation index generator runs, it performs the following tasks:

  1. The generator downloads the latest documentation data from the MicrosoftDocs organization on GitHub.
  2. It parses the markdown data to retrieve the names and descriptions of clipboard formats.
  3. If the description includes a link with text that is all uppercase (usually this indicates it is a struct), it parses the relative path and downloads the markdown data for that page.
  4. Once that markdown data is found, the generator parses it to strip the header and other unneccessary information.
  5. It repeats this process for every other format.
  6. Once all the formats are done, it serializes the data into a JSON file that can be loaded later.

This process does require an internet connection, but it ensures the documentation is always up-to-date. You can manually initiate the generator by clicking Update Documentation on the Help menu.
