Using Clipboard Snapshots

A clipboard snapshot allows you to save the current data in the clipboard to view later. To save a snapshot:

  1. On the File menu, select Save Snapshot.
  2. Browse to the location where you want to save the snapshot.
  3. Choose a name for your snapshot. To create a compressed snapshot, use the .CSNAPZ file extension instead of the regular .CSNAP extension.
  4. Click Save to save the snapshot.

To open a snapshot:

  1. On the File menu, select Open Snapshot.
  2. Browse to the location where the snapshot file is stored.
  3. Select the snapshot file and click Open.

Note

  • You can change the compression level for .CSNAPZ files in the ClipboardEdit Settings. On the File menu, click Settings.
