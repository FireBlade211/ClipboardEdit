The documentation index

The main part of the ClipboardEdit Documentation System is the documentation index.

The index is usually located in %LOCALAPPDATA%\ClipboardEdit\FormatDocData.json and stores information about the names of clipboard formats, as well as their short description, and most importantly: their struct documentation markdown data.

The file uses the JSON format and is stored as a JSON array which usually looks like this:


[
     {
         "Name": "CF_TEXT",
         "Description": "A short description of the format.",
         "AdditionalStructDocs": "Additional documentation for the structure that may be contained in this format. Markdown is supported. Leave empty to specify a struct does not exist for this format or does not have documentation available."
     },
     {
         "Name": "CF_UNICODETEXT",
         "Description": "A short description of the format.",
         "AdditionalStructDocs": "Additional documentation for the structure that may be contained in this format."
     }
]

The information stored in this index file is usually created by the documentation index generator.
